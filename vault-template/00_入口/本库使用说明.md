# 本库使用说明

这是一个以 Obsidian 为数据底座、以 Agent 路由为处理层的任务管理与知识管理模板。

## 每天只打开这三个

- `10_Daily/当天日期.md`
- [[../00_Inbox/收件箱]]
- [[../70_Dashboards/Inbox处理台]]

## 新版 Agent 工作流

1. 捕获：任何内容先进 Inbox。
2. 路由：让 Agent 按协议给分类和下一步建议。
3. 决策：你接受、修改、合并、新建项目或丢弃。
4. 写入：Agent 根据确认结果更新 Daily、Project、Area、Goal、Resources、Reviews 或 AI_Memory 候选。
5. 执行：你只看 Daily 和当前项目下一步。
6. 复盘：晚上或周日沉淀经验。
