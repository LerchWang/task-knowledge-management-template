# Project Hub 大项目结构

大项目会自然跨 Goals、Projects、Areas、Resources、Reviews 和 AI Memory，所以必须有一个主线入口。

推荐每个大项目使用独立文件夹：

```text
项目名
├── README.md
├── 00_项目总览.md
├── 01_任务池.md
├── 02_执行日志.md
├── 03_决策记录.md
├── 04_资料索引.md
├── 05_复盘记录.md
├── 06_版本记录.md
├── 10_沉淀
└── 子项目
```

## 规则

- 父项目管主线。
- 子项目管阶段工作。
- 交付物跟随子项目。
- Goals / Areas / AI Memory 只挂链接和摘要，不保存完整项目正文。
