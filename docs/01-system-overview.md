# 系统总览

这个系统把任务管理、知识管理、复盘和 AI 长期记忆放进同一个 Obsidian Vault。

## 目录层级

```text
00_Inbox       临时收集
00_入口         使用说明和教程
10_Daily       每日执行入口
20_Goals       目标与计划
30_Projects    项目推进
40_Areas       长期责任领域
50_Resources   资料和教程
60_Reviews     复盘和异常处理
70_Dashboards  聚合视图
80_AI_Memory   给 AI 的长期摘要
90_Templates   模板
99_Archive     归档
```

## 每天只看哪里

高频入口只有三个：

- Daily Note：今天做什么。
- Inbox：先接住临时想法。
- 任务总览：桌面端查看到期和未完成任务。

## 核心工作流

```text
捕获 → 分流 → 执行 → 复盘 → 沉淀/归档/删除
```
