# Task & Knowledge Management Template

一个以 Obsidian 为核心的中文个人任务管理、知识管理和 AI 长期记忆模板。

这个仓库适合想要建立低摩擦执行系统的人：任务、项目、复盘、资料、长期领域和 AI 记忆都在同一个 Obsidian Vault 中协作，但每天只需要看 Daily Note。

## 核心思想

- Inbox 是入口，不是家。
- Daily 是当天战场，不是仓库。
- Goals 管方向和衡量标准。
- Projects 管有完成线的阶段交付。
- Areas 管长期责任和维护规则。
- Resources 放资料。
- Reviews 放经验。
- AI Memory 放给 AI 的稳定摘要。
- Archive 放完成后的历史。
- 删除是保持系统轻盈的必要动作。

## 适合谁

- 个人单兵作战为主，偶尔协作。
- 需要中文任务管理与知识沉淀。
- 想把 Obsidian 用作项目记录、复盘和 AI 记忆核心。
- 不想在复杂工具和形式维护上浪费太多精力。

## 快速开始

1. 下载或 clone 本仓库。
2. 复制 `vault-template` 文件夹，改名成你的 Obsidian Vault。
3. 用 Obsidian 打开这个文件夹。
4. 从 `00_入口/本库使用说明.md` 开始。
5. 每天只打开 `10_Daily/当天日期.md`、`00_Inbox/收件箱.md` 和 `70_Dashboards/任务总览.md`。

## 仓库结构

```text
docs/                 方法说明
vault-template/        可直接打开的 Obsidian Vault 模板
templates/             单独可复制的项目/子项目/日记模板
examples/              示例项目
assets/diagrams/       生命周期和结构图
tools/                 可选辅助脚本
```

## 推荐 Obsidian 插件

第一阶段：

- Calendar
- Tasks
- Templater

第二阶段：

- Dataview
- QuickAdd
- Excalidraw

插件只是加速器。即使不安装插件，Markdown 版本仍然可以运行。
